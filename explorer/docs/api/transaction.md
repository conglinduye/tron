## 查询交易信息
- url:/api/transaction
- method:get

input:param
```param
&limit=40       //每页40条记录
&count=true     //是否返回总数
&start=0        //记录的起始序号
&sort=-timestamp       //按照时间戳倒序排列

eg: http://18.216.57.65:20110/api/transaction?sort=-number&limit=40&start=0
```
output:json

- method:get

input:param
